## Commands for starting the application:
### Back-End
- cd back-end
- npm install (if the node modules are not avilable)
- node index.js
- The back-end side will start on localhost://4001 port



### Front-End
- cd front-end
- npm install (if the node modules are not avilable)
- npm start
- The front-end side will start on localhost://3000 port
